Server-side Description

The server-side of Artify is a Node.js and Express application that handles RESTful APIs for managing artworks and user data. It interacts with a MongoDB database to store artworks, user information, likes, and favorites. The backend provides secure endpoints for CRUD operations, user authentication, and favorite management, enabling the client-side to fetch and update data efficiently.

Key Features:

CRUD operations for artworks (Create, Read, Update, Delete)

User authentication and session management

Handling likes and favorites for artworks

MongoDB database integration with Mongoose

RESTful API endpoints for seamless client-side integration
